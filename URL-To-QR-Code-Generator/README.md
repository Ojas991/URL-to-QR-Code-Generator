# URL to QR Code Generator

A simple web application that converts any valid URL into a QR Code instantly using HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://url-to-qr-code-generator.onrender.com

## 📂 GitHub Repository

https://github.com/Ojas991/URL-to-QR-Code-Generator

## ✨ Features

- Convert any URL into a QR Code
- Simple and responsive user interface
- Instant QR code generation
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- Render

## 📁 Project Structure

```
URL-To-QR-Code-Generator/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter a URL.
4. Click **Generate QR Code**.
